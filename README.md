# Soundboard Application

A browser-based soundboard that allows you to upload and play sounds with customizable buttons.

## Features

- Upload custom sounds to buttons
- Play sounds with mouse clicks or keyboard shortcuts
- Adjust volume for each sound individually
- Delete sounds from buttons
- Keyboard shortcuts for quick access

## How to Use

1. Open `soundboard.html` in any modern web browser
2. Click on any button to upload a sound file (MP3, WAV, OGG, etc.)
3. After uploading, click the button to play the sound
4. Hover over a button with a sound to adjust its volume
5. Use the delete button (✕) to remove a sound from a button
6. Use keyboard shortcuts to play sounds:
   - Keys 1-9 for buttons 1-9
   - Keys Q, W, E for buttons 10, 11, 12
   - Delete/Backspace to remove sound from selected button

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge)
